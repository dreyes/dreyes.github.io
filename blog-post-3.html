Is CSS a programming language? I believe it's NOT!

March 20, 2020

share on:

Image credits to Unsplash.com

I recently started a Front-End Developer path, hence the reason for this blog.

Any Web Developer will tell you, if you want to learn to program for the web there's no better place to start than HTML and CSS.

Even though common sense will tell you CSS is a programming language, here's what I've got to say:

It's NOT! CSS is harder than any other programming language!

For some, it is a forgiving language. Why? It will allow you to make mistakes and syntax is not your biggest worry.

But CSS is not only about the programming; you must also have a sense of design. CSS is a way to describe what an interface should look like!

Although I'm still on my learning process, I believe I'm starting to get-a-knack of it.

And for those of you who haven't adventured in the programming world, why not give it a try? You might as well end up landing your next remote job!
